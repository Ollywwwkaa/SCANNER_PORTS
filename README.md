# Основная суть проекта:

сетевой сканер, который проверяет, какие порты на компьютере или сервере открыты для подключения, используя многопоточность для быстрого сканирования.

# Что делает программа?

"Стучится" в двери - пробует подключиться к портам (как номерам дверей на сервере)

Cмотрит, какая дверь открыта - если подключился → порт открыт

Угадывает, кто за дверью - определяет сервис (HTTP, SSH, FTP и т.д.)

Быстро работает - использует много "работников" (потоков) одновременно

# Что МОЖЕТ этот сканер:

Определять сервисы по стандартным портам

Работать быстро благодаря потокам

Сохранять результаты в файл

  

# Вывод

для проверки сетевой доступности - простой, быстрый и полезный инструмент, который показывает, какие сервисы доступны на компьютере или сервере, используя современные подходы к параллельной обработке и удобный интерфейс командной строки.

  
  

#Пример использование сканера 1.

#Сканируем IP-адрес с диапазоном портов 1-1000, показывает все порты то есть не только открытые (-a)

#Добавляем сортировку от большего к меньшему (--desc) и результат сохраняем в отдельный файл (-о)

```python
python scanner.py 192.168.1.1 -p 1-1000 -a --desc -o sorted_res.txt

```

КРАТКО:

'-p', '--ports', default='1-1000',  # Аргумент для портов

'-t', '--threads', type=int, default=50,  # Аргумент для потоков

'-s', '--service', action='store_true',  # Флаг определения сервисов

'-a', '--all', action='store_true',  # Флаг показа всех портов

'--timeout', type=float, default=2.0,  # Аргумент для таймаута

'-o', '--output', type=str,  # Аргумент для сохранения в файл

'--desc', action='store_true',  # Флаг для сортировки по убыванию
